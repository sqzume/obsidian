### インターネット接続確認
```
# ping archlinux.jp
```

- コマンドは`Ctrl + C`で止めることができる
### パーティション設定
ディスクのデバイス名を調べる
```
# lsblk
```

gdiskの起動
```
# gdisk /dev/<デバイス名>
```
gdiskのコマンド一覧

EFIシステムパーティションの作成
```gdisk
Command (? for help): n
Partition number: <Enter>
First sector: <Enter>
Last sector: +512M
Hex code or GUID: ef00
```

Arch Linux用パーティションの作成
```gdisk
Command (? for help): n
Partition number: <Enter>
First sector: <Enter>
Last sector: <Enter>
Hex code or GUID: <Enter>
```

現在のパーティションの確認
```gdisk
Command (? for help): p
```

設定した
**Tips:*
- パーティションテーブルのフォーマットはGPT
