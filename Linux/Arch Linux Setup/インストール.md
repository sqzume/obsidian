### パーティショニング
ディスクのデバイス名を調べる
```
# lsblk
```

gdiskの起動
```
# gdisk /dev/<デバイス名>
```
gdiskのコマンド一覧

EFIシステムパーティションの作成
```gdisk
Command (? for help): n
Partition number: <Enter>
First sector: <Enter>
Last sector: +512M
Hex code or GUID: ef00
```

Arch Linux用パーティションの作成
```gdisk
Command (? for help): n
Partition number: <Enter>
First sector: <Enter>
Last sector: <Enter>
Hex code or GUID: <Enter>
```

現在のパーティションの確認
```gdisk
Command (? for help): p
```

設定したパーティションの書き込み
```gdisk
Command (? for help): w
Do you want to proceed? (Y/N): y
```

**Tips:*
- パーティションテーブルのフォーマットはGPT
### フォーマット
EFIシステムパーティションをFAT32でフォーマット
```
# mkfs.vfat -F32 /dev/<EFIシステムパーティション>
```

Arch LinuxパーティションをBtrfsでフォーマット
```
# mkfs.btrfs /dev/<Arch Linuxパーティション>
```

確認
```
# lsblk --fs
```
